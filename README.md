# Demo for Bold Tech Company

Bold Server, build on [LoopBack](http://loopback.io).

## Next steps:

Change directory to your app

    $ cd bold

Install dependences

    $ npm install

Test

    $ gulp test

Run the app

    $ node . (Use the explorer - http://localhost:3000/explorer)
